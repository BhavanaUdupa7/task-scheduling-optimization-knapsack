# Task Scheduling Optimization using Dynamic Programming

This project solves the task scheduling problem using the 0/1 Knapsack algorithm. It helps in selecting the most important tasks within limited available time to maximize overall productivity.

## Problem Statement
Given a set of tasks, each with a duration and priority, the goal is to select tasks such that:
- Total duration does not exceed available time
- Total priority (productivity) is maximized

## Approach
- Implemented using Dynamic Programming (0/1 Knapsack)
- Backtracking is used to identify selected tasks
- Visualization is done using Matplotlib

## Features
- User input for tasks, durations, and priorities
- Optimal task selection
- Productivity maximization
- Graphical visualization of selected tasks

## Technologies Used
- Python
- Dynamic Programming
- Matplotlib

## Output
- Maximum achievable priority
- List of selected tasks
- Bar chart visualization of selected tasks
